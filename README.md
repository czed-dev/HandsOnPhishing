# Hands-On Phishing Course

Learning materials and scripts from Tyler Ramsbey's Hands-On Phishing course.

## Course Overview

Practical training for red teamers and security professionals on building and executing phishing campaigns using real-world techniques and tools.

## Topics Covered

- Lab environment setup
- Domain acquisition and configuration
- Email infrastructure setup
- GoPhish and Evilginx2 implementation
- Bypass techniques for modern defenses
- Session cookie theft
- Email and phone call pretext combinations

## Repository Structure

- `scripts/` - Custom tools and automation scripts
- `notes/` - Course notes and documentation
